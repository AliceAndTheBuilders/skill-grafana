# opsdroid skill hello

A skill for [opsdroid](https://github.com/opsdroid/opsdroid) to respond to hello and goodbye messages.

## Requirements

None.

## Configuration

None.

## Usage

#### `hello`

Says hello to the user.

> user: hello
>
> opsdroid: Hi user

#### `goodbye`

Says goodbye to the user.

> user: bye
>
> opsdroid: Bye user
